# Tuk-Tuk Tracking API

Real-Time Three-Wheeler (Tuk-Tuk) Tracking & Movement Logging System for Sri Lanka Police

**Student ID:** 16114151  
**Module:** Web API Development  
**Batch:** 24.2P

## Overview

A centralized RESTful API for Sri Lanka Police to track registered three-wheelers (tuk-tuks) in real-time using GPS pings.

## Tech Stack

- **Runtime:** Node.js (ES6+)
- **Framework:** Express.js
- **Database:** MySQL + Sequelize ORM
- **Auth:** JWT (Access + Refresh tokens)
- **Docs:** Swagger UI
- **Logging:** Winston + Morgan
- **Validation:** Joi
- **Error Monitoring:** Sentry
- **Deployment:** Railway
- **Containerization:** Docker
- **CI/CD:** GitHub Actions
